# 示例音频目录 / Example Audio Folder

这个目录用于放置你自己的授权示例音频。为了避免公开仓库分发可能受版权保护的音乐，本仓库默认不提交 `.mp3`、`.wav`、`.flac` 等音频二进制文件。

This folder is reserved for your own authorized example audio files. To avoid redistributing potentially copyrighted music, this repository does not commit `.mp3`, `.wav`, `.flac`, or other audio binaries by default.

## 推荐文件名 / Recommended filenames

```text
examples/audio/original_song.mp3
examples/audio/covered_by_my_voice.mp3
```

## 公开发布提醒 / Publishing reminder

如果你要把示例音频提交到公开 GitHub 仓库，请先确认你拥有版权或已获得授权。

Before committing example audio to a public GitHub repository, make sure you own the rights or have explicit permission.
